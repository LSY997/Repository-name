# 乐新电子 - 电商绩效总控台

## 项目介绍

这是一个基于 Streamlit 的电商绩效管理系统，用于管理运营、售前、售后等部门的绩效数据。系统提供完整的数据录入、分析和可视化功能。

## ✨ 功能特性

### 👤 角色权限管理
- **总监**：查看全店绩效总览、销售排行、ROI分析
- **主管**：数据录入、部门数据分析
- **操作员**：运营数据录入、个人绩效查看

### 📝 数据录入模块
- **运营数据**：访客数、浏览量、销售额、订单数、广告花费
- **售前接待**：接待量、意向客户、咨询转化率
- **售后跟踪**：退货申请、纠纷单数

### 📊 可视化分析
- 💰 销售占比分析（饼图）
- 📈 转化率趋势（折线图）
- 📦 售后指标对比（柱状图）
- 🎯 ROI 计算分析

### 💾 数据持久化
- JSON 文件存储
- 日期分类管理
- 自动备份功能

### 🔐 身份验证
- 用户登录认证
- 会话管理
- 权限级别控制

## 🚀 ���速开始

### 环境要求
- Python 3.8+
- pip

### 安装依赖
```bash
pip install -r requirements.txt
```

### 配置环境变量
```bash
cp .env.example .env
```

### 运行应用
```bash
streamlit run app.py
```

应用将自动在浏览器中打开 `http://localhost:8501`

## 📱 主要页面说明

### 登录页面
- 用户身份验证
- 角色权限识别

### 数据录入页面（操作员/主管）
- **运营数据 Tab**：录入日常运营指标
- **售前接待 Tab**：记录售前数据
- **售后跟踪 Tab**：跟踪售后情况

### 总监总控台（总监）
- **关键指标卡**：展示总销售额、订单数、访客数、ROI
- **销售占比分析**：各团队销售贡献度
- **售前转化趋势**：转化率变化曲线
- **售后风险预警**：退货和纠纷监控

## 🔄 数据流程

```
用户登录 → 身份验证 → 根据角色显示相应功能
  ↓
数据录入 → 验证 → 保存到 JSON → 加载数据库
  ↓
总监查看 → 数据聚合 → 图表展示 → 决策分析
```

## 💡 使用建议

1. **日常操作**：操作员每日录入运营数据
2. **质量监控**：主管定期审查部门数据
3. **战略决策**：总监根据总览数据制定策略

## 🐛 故障排查

### 问题：无法加载数据
**解决方案**：检查 `performance_data.json` 文件权限

### 问题：登录失败
**解决方案**：确认用户名和密码完全匹配（区分大小写）

### 问题：图表不显示
**解决方案**：确保已安装 Plotly，运行 `pip install --upgrade plotly`
